# OSSIQN TV - AutoStream

Sifir ayar, API derdi olmayan, "Kur & Izle" mantigiyla calisan acik kaynak dizi, film, anime ve canli spor yayin scripti.

## Ozellikler

* **Sifir Yapilandirma:** Veritabani ayarlariyla veya tablo olusturmakla ugrasmaniza gerek yok. Sistem ilk calismada veritabanini ve tablolari otomatik olarak kendi olusturur.
* **Otomatik Icerik Cekme:** Film, dizi ve animeler kendi icindeki ozel altyapi sayesinde aninda cekilir. Harici API key aramaniza gerek kalmaz.
* **Modern ve Hizli Arayuz:** Tamamen mobil uyumlu, karanlik mod (dark mode) destekli, hizli ve kesintisiz arayuz.
* **Sohbet & Etkilesim:** Canli spor yayinlarinda izleyicilerin anlik sohbet edebilecegi, sunucuyu yormayan JSON tabanli chat modulu.
* **Temiz Altyapi:** PHP ile yazilmis, tertemiz, hizali ve acik kaynak kodlama. Sifreleme (ioncube vb.) kesinlikle yoktur.

## Kurulum

1. Repoyu bilgisayariniza indirin veya zip olarak cikartin.
2. Tum dosyalari sunucunuzun `public_html` dizinine veya localhost klasorunuze yukleyin.
3. Tarayicinizdan site adresinize (veya localhost'a) giris yapin.
4. Sistem gerekli veritabani kurulumlarini otomatik yapacak ve kullanima hazir hale gelecektir. Arkaniza yaslanin ve izlemeye baslayin.
